# Twitch Chat Voicebot

## Основные функции

-  Мониторинг чата в реальном времени: Подключение к IRC Twitch для прослушивания новых сообщений.
-  Использование TTS-движка (Silero TTS) для озвучивания сообщений.
-  Логирование всех действий (удалённые сообщения, ошибки)
-  Работа со списком пропуска сообщений


# Запуск бота

Копирование репозитория

```bash
git https://github.com/justforgetmeeee/twitch_chat_voicebot.git
cd twitch_chat_voicebot
```

Установка зависимостей
```bash
pip intall -r requirements.txt
npm i
```
Запуск бота
```bash
node main.js
```
